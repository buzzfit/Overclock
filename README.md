# OVERCLOCK (Web + Desktop)

**Play in your browser:** (after first CI run)
`https://<your-username>.github.io/<your-repo>/`

Controls: Move = WASD / Arrows • Fire = LMB / SPACE • Overclock = SHIFT • Pause = P  
Tip: Click once to enable audio (browser autoplay policy).

**Downloads (optional):** see the **Releases** tab for Windows/macOS/Linux zips.

## Dev notes
- Single-file Pygame game under `game/main.py`.
- Web builds are produced with **pygbag** and auto-deployed to **GitHub Pages**.
- Desktop builds are produced with **PyInstaller** via `release.yml`.
